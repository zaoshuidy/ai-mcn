# AI MCN商业内容Agent原型

基于真实小红书达人内容证据，完成达人研究、风格分析、商业脚本生成、自然化、食品广告合规审核和分镜输出，并将流程整理为可复用Skill与Agent原型。

## 背景与方案

业务痛点是品牌内容缺少可追溯的达人、卖点和合规证据。解决方案是：品牌Brief → 达人研究 → 风格研究对象选择 → 三视频风格蒸馏 → 商业脚本 → 脚本自然化 → 事实回归 → 食品广告合规 → 分镜 → 人工审核 → 飞书Publisher接口。

## 核心数字

v2 已评分 24 位；严格候选 2 位；Kelly 三视频分析；5 个 Skill；1 条轻醒内容链；19 镜/45.0 秒；26 个组件登记；21 个 ≥1000 星组件；组件准入 47 项测试；Stage 2 为 95 PASS；全量 pytest 659 通过、4 项环境依赖失败。

## 导航与阅读路径

5分钟：项目概览 → 达人研究 → 最终脚本 → 合规报告 → 验证报告。

10分钟：再加入 Kelly 三视频、分镜、Agent/Skill、高星参考、飞书正文和限制说明。

快速验证：`python -m pytest tests/test_component_admission.py -q`、`python scripts/validate_stage_2.py`、`ruff check .`。

项目边界：不声称签约、自动发布或正式投放；品牌事实与发布均需人工审批。
